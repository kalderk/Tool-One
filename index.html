Tool 1 -- Vehicle Emissions Calculator

Use the sliders below to set the number of cars and trucks passing through a given area per hour. This tool allows the user to set a number of cars passing by on the highway and see the amount of pollutants that result. The four pollutant outputs tracked are CO, VOCs, NOx, and PM, users can see the mass produced of each of these pollutants by the cars passing by per hour. The rates at which these pollutants are produced is calculated from the flowchart and are in per mile values (assumed speed of cars is 60 mph). These values can be applied to the mile section of highway this airshed is looking at along I-75 in Dallas, Texas.

Reflection

With this tool, the biggest assumption is that the only emissions that are affecting the area are the ones in that one mile segment. Having this assumption allows for the calculations of the air pollutants to be done easier along with the constant 60 mph speed assumption since the pollutants emitted in one minute can be found. This is important as it is the baseline for all of the calculations the tool does since the calculated values of emitted pollutants is on a per car per minute basis. What makes or breaks the tool is the values for our air pollutants, the calculations done to determine these donâ€™t seem perfect and a better look into these values might lead to a more accurate answer. I think another thing I could work to improve upon is making the tool make more sense. Right now it is simply showing number values for all of the pollutants with not much context provided on what these pollutants do. I think maybe adding some color coding or symbols to show dangerously high levels or the range of concentrations better would be helpful to decipher the values beyond just a number on the screen. As for AI, I used VS code to write most of the HTML script along with using some tutorial pages I found online, the embedded copilot feature was a big help in terms of debugging the code. Additionally I used Claude to further debug and help me write some other features of the webpage (as well as help me with some of the features on github itself). The one place I had to intervene with the AI was on the calculations for the pollutant emissions, it essentially pulled out random values for the tables which threw me off at first because I thought I had given it these values, but I eventually did the calculations by hand to determine the outputs.
